## Утверждённая структура лендинга

1. Верхняя бегущая строка (фиолетовая) — достижения наставников  
2. Hero — джог CDJ + DJ/PLACE + слоган + кнопки + 4 цифры  
3. Нижняя бегущая строка (тёмная)  
4. Наставники — 4 карточки 2×2, тап открывает fullscreen overlay снизу вверх  
5. Пульт — "Попробуй диджеинг прямо сейчас"  
6. Курсы — 3 карточки  
7. Студия — фото + оборудование (без повтора цифр из hero)  
8. Отзывы — 3 штуки  
9. FAQ  
10. Финальный CTA "ТВОЯ СЦЕНА ЖДЁТ"  
11. Форма записи (модальное окно)

## Принципы работы

**Маркетинг:** никаких выдуманных цифр, ложного дефицита, завышенного числа выпускников. Только верифицируемые факты. Сила позиционирования — реальные достижения конкретных наставников.

**Код:** изменения через точные `str_replace`-блоки, не полная перезапись файлов.  
**Визуал:** сначала мокап, потом код.  
**Ответы:** краткие, но без потери важного. Критика идей приветствуется.  
**Язык:** русский.

## Правила для текстов (обязательно)

**Тире запрещены** — ни в одном тексте, ни в заголовке, ни в подписи.  
Вместо тире: перестраивай предложение, используй двоеточие, скобки или разбивай на два предложения.  
Перед отдачей любого текста проверь каждую строку на наличие тире.

- Cloudflare proxy несовместим с Vercel (Vercel автоматически отключает его). Для текущего сетапа правильный режим — DNS-only.
